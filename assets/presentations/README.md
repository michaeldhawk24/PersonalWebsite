# Project presentations

Do **not** paste the PDF contents into this `README.md` file. Keep the README as instructions, and place the actual PDF file in this same folder.

For the Stock Oil Pump Flow Rate Testing project:

1. Export the PowerPoint, Keynote, or Google Slides presentation as a PDF.
2. Rename the exported file exactly:

   `stock-oil-pump-flow-rate.pdf`

3. Move or upload the renamed PDF into this folder:

   `assets/presentations/`

4. The final file path should be:

   `assets/presentations/stock-oil-pump-flow-rate.pdf`

5. The project card in `index.html` already links to that path, so the website button will work after the PDF is added.

If you also want this README to show a clickable link to the uploaded PDF, use this Markdown:

```md
[View the Stock Oil Pump Flow Rate presentation](./stock-oil-pump-flow-rate.pdf)
```

After the PDF is uploaded, this link will point to the file sitting beside this README.
