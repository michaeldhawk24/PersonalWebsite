# Project presentations

Add project presentation PDFs to this folder so portfolio links can point to a stable, repo-relative path.

For the Stock Oil Pump Flow Rate Testing project:

1. Export the PowerPoint or Keynote presentation as a PDF.
2. Rename the exported file to `stock-oil-pump-flow-rate.pdf`.
3. Place it in this folder so the final path is:

   `assets/presentations/stock-oil-pump-flow-rate.pdf`

The project card in `index.html` already links to that path.
