# Personal Website

A modern, responsive personal portfolio website for a software engineer.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Hero Section**: Eye-catching introduction with call-to-action
- **About Section**: Brief bio and professional background
- **Projects Showcase**: Display featured projects with tech stack
- **Skills Section**: Organized list of technical skills
- **Contact Section**: Easy ways to get in touch
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Animations**: Subtle fade-in and slide animations

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling with responsive design
- `script.js` - JavaScript for interactivity and animations

## Getting Started

1. Clone this repository
2. Customize the content:
   - Update your name and title in the hero section
   - Replace project descriptions with your own projects
   - Update skills list with your technologies
   - Add your contact information (email, LinkedIn, GitHub, etc.)
3. Open `index.html` in your browser

## Customization

### Colors
Edit the CSS variables in `styles.css` to change the color scheme:

```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00d4ff;
    /* ... other colors ... */
}
```

### Content
- Replace placeholder text with your actual information
- Update project links to point to your GitHub repositories
- Add your actual contact information

### Fonts
The site uses 'Segoe UI' as the primary font. You can modify this in `styles.css`.
